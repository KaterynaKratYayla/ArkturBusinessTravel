Про Компанію

ТОВ «АРКТУР» - це провідна українська компанія з обслуговування подорожей та заходів для MICE та корпоративного сегменту з головним офісом в Києві та представництвом у Львові, яка була заснована в 1994 році. ТОВ «АРКТУР» надає обслуговування в Україні та за кородоном (під запит) на найвищому рівні, згідно міжнародних норм та стандартів. Наша команда складається з професіоналів, що вільно володіють декількома іноземними мовами та мають 10-25 річний досвід роботи у галузі туризму та обслуговування корпоративного та конференц сегменту.

Нами було створено успішно діючу багатофункціональну платформу онлайн бронювання послуг з розміщення у готелях по всієї Україні по найбільш конкурентним тарифам. Система має інтерфейси як для прямого клієнта , HotelsUkraine.Travel, так і для агентів та корпоративних клієнтів, та надає можливість її користувачам:

  • Отримати доступ до нашого портфоліо, що включає більш ніж 250 готелів, законтрактованих за прямими договорами, по всієї Україні з миттєвим підтвердженням запиту;
  • Забронювати транспортне та екскурсійне обслуговування з автоматичним підтвердженням замовлення;
  • Провести інтеграцію/обмін інформацію про наші туристичні продукти на мові XML для систем бронювання та внутрішніх систем корпоративних клієнтів;
  • Клієнтська підтримка 24/7;
  • Оплатити за бронювання в онлайн режимі з використанням кредитної картки або отримати онлайн рахунок для банківського перерахування коштів;

Обслуговування MICE та Корпоративного сегменту в Україні та за кордоном, яке включає :

  • Розміщення та транспортне обслуговування;
  • Авіа та залізно-дорожні квітки;
  • Організацію та обслуговування конференцій та семінарів;
  • Послуги з організації корпоративних та інсентів поїздок;
  • Мотиваційні Teambuilding програми , такі як :
    • Worldwide : Quests / 4×4 Trips / Sailing / Go Kart / Paintball activities / Segway or bicycle activities , River Cruises ;
    • Ukraine specials : Cooking Master Classes / Ballet Performance and Opera Lovers programs / Ukrainian Wedding activities / Chocolate Master Classes / Ukrainian Coffee Tour / Chernobyl Tour;

ТОВ «АРКТУР» є членом IATA та Асоціація Ділового Туризму України (BTA – Ukraine) .

Ділові Подорожі : Послуги з розміщення

ТОВ «Арктур» надає послуги розміщення у будь якому готелі України.

Зокрема з 255ми готелями України у компанії «Арктур» підписані прямі договори зі знижками та квотами / «м’якими» квотами місць.

Майже всі 255 готелів доступні для бронювання в онлайн режимі в системі бронювання «Арктур» під закритим доступом з автоматичним підтвердженням бронювання та за конкурентними тарифами, які постійно оновлюються в системі з боку готелів самостійно

Усі наші готелі України в повному обсязі є доступними для перегляду та бронювання через ArkturB2B - нашу Б2Б систему онлайн бронювання.

Наші Технологічні Рішення

Розроблена нашою командою програмістів система СМАРТ є багатофункціональною платформою , яка має наступні інтерфейси :

  • Інтерфейс постачальника : розроблений для готелів, які управляють своїми тарифами та квотами в нашій системі напряму у СМАРТ;
  • Інтерфейс для систем управління каналами – channel managers : один з різновидів інтерфейсу для постачальників, що дозволяє їм управляти продажом у системі СМАРТ опосередковано, через агрегаторів управління каналами (наприклад, системи YieldPlannet, Travellene, SystemOtel-Туреччина), з якими наша система СМАРТ інтегрована;
  • Інтерфейс для агента та корпоративного клієнта : використовується нашими партнерами для бронювання туристичних послуг, зокрема готелів в Україні та деяких готелів Туреччини, під закритим доступом у режимі реального часу;
  • Інтерфейс для індивідуального клієнта (тобто B2C) – hotelsukraine.travel з можливостю бронювання готелів , трансферів, екскурсій по Україні з оплатою кредитною карткою через LiqPay ПриватБанку;
  • Інтерфейс для платформ продажу туристичних услуг, які мають можливість інтегрувати модуль нашої системи на отримати миттевий системний доступ по всієї бази наших готелів.
